# 📊 Jharkhand Business Analysis Dashboard

An interactive Business Analytics Dashboard built using **Python**, **Streamlit**, **Plotly**, and **Folium** to analyze business distribution, opportunities, and market trends across Jharkhand.

The project combines data analysis, visualization, and geospatial mapping to provide meaningful insights into different business categories and their locations.

---

## 🚀 Features

- 📈 Interactive Business Analytics Dashboard
- 🗺️ Business Location Mapping using Folium
- 📊 Dynamic Charts with Plotly
- 🔍 Category-wise Business Analysis
- 📍 District-wise Business Distribution
- 📉 Business Opportunity Visualization
- ⚡ Fast and Interactive Streamlit Interface

## ▶️ Running the Project

### Clone the Repository

```bash
git clone https://github.com/bhavya-aggarwal011/Jharkhand-Business-Analysis.git
```

### Navigate to Dashboard Folder

```bash
cd Jharkhand-Business-Analysis/dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Dashboard

```bash
streamlit run app.py
```

or

```bash
python -m streamlit run app.py
```
